**太虚引擎（VOID AR)**是由成都米有网络科技有限公司自主研发的，集成于Unity3d实现增强现实的SDK开发包,目前Alpha-0.02版本支持图片以及手绘图像的增强现实。


* 
**开发环境需求: **

 操作系统 Win7 Win8 Win10 MacOS 10.x 
 Unity3d  32位 4.x--5.x 

* 
**跨平台支持情况: **

 Windows ：支持 
 
 Ios：支持 
 
 Android：暂不支持（后续版本支持） 
 
 <br>
 太虚引擎（VOID AR)具有以下功能特点：
 
 **手绘识别：**可以识别手绘简笔画。
因为现在所有SDK只能对印刷目标进行识别，所以手绘识别是太虚引擎的特色功能。<color=#FF3030>啦啦啦</color>
